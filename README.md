# ✦ VerifyAI – Deepfake Detection System

🚀 **VerifyAI** is a modern web-based application that detects whether an image is **real or AI-generated (deepfake)** using advanced AI models.
It provides a sleek UI and near real-time analysis to help users verify digital authenticity.

---

## 🔍 Overview

With the rapid rise of AI-generated content, identifying fake or manipulated images has become critical.
VerifyAI leverages **AI-powered image forensics** to analyze uploaded images and classify them as **Authentic** or **AI Generated**, along with a confidence score.

---

## ✨ Features

* ⚡ **Fast Detection** – Analyze images in seconds
* 🎯 **High Accuracy** – Smart AI-based prediction system
* 🖼️ **Image Upload & Preview**
* 📊 **Confidence Score Visualization**
* 🌙 **Dark/Light Mode UI**
* 🎨 **Modern Glassmorphism Design**
* 🔐 **Secure Processing (No permanent storage)**
